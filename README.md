# T&O Executive Dashboard

An interactive dashboard built with **Next.js**, **Tailwind CSS**, and **Chart.js**.  
It displays:
- Spend by Program
- Cost Category Distribution
- Spend by Region

## Features
- Responsive design using Tailwind CSS
- Dynamic data fetching via Next.js API routes
- Interactive charts using Chart.js
